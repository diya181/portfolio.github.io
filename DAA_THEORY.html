Welcome to the Course!

Course Info

This course explores the design and analysis of algorithms, covering fundamental data structures, algorithm efficiency, and real-world applications.

Data Structures and Algorithms We Studied

  • Arrays, Linked Lists, Stacks, Queues
  • Trees (Binary Search Trees, AVL Trees)
  • Graphs (Representations, Traversals)
  • Sorting Algorithms (Merge Sort, Quick Sort)
  • Searching Algorithms (Binary Search)

Real-Time Applications

These concepts are applied to real-world problems like optimizing warehouse management, network routing, and search engine indexing.

Assigned Business Cases (Members Working On)

  • Member 1: Listing of all preferences of users: This case involves creating a system that can collect and display user preferences, such as favorite destinations, activities, or types of accommodations. The system should be able to insert new preferences and update existing ones.
  • Member 2: Quick visits: In this case, the system recommends places to visit based on their proximity to the user's current location
  • Member 3: Listing all the eco-traveling options: This case involves creating a system that lists eco-friendly transportation options, such as walking, biking, or taking public transport.
  • Member 4: Shop local: This case promotes local culture by sorting shops alphabetically or by category.
